AnyRail with JMRI

The AnyRail layout planning software can now export a layout design as a JMRI panel file that can be imported to Layout Editor. See the AnyRail documentation.

Hints for using AnyRail's export files

When you do the export from AnyRail, it creates two files: An XML panel file and an image that provides the background. You should have AnyRail export those directly to the JMRI profile directory so that they stay linked together.

The export file is a minimal PanelPro data file. It contains a turnout table, a block table and the layout editor panel. By default, the user names for the turnout table and the block table entries will be the panel internal name. If the AnyRail labels have been used for turnouts and sections (JMRI blocks) the label value will be used for the user name. This will help understand the layout since the internal names are very cryptic.

Since the exported PanelPro file is minimal, most people will need to do additonal editing using Layout Editor. For people new to JMRI, or new to PanelPro and Layout Editor, reading the Getting Started with PanelPro tutorial is recommended. This will provide an introduction to PanelPro tables and creating a simple Layout Editor panel.
