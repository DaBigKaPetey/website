AnyRail with JMRI

The AnyRail layout planning software can now export a layout design as a JMRI panel file that can be imported to Layout Editor. See the AnyRail documentation.

Hints for using AnyRail's export files

When you do the export from AnyRail, it creates two files: An XML panel file and an image that provides the background. You should have AnyRail export those directly to the JMRI profile directory so that they stay linked together.
